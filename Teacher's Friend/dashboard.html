வழிகாட்டுதல்கள்

வகுப்பறை பகுதியை சிறப்பாக மேலாண்மை செய்ய பின்வரும் வழிகாட்டுதல்களை கடைபிடிக்கவும்:


1. Data input button யை கிளிக் செய்து அதில் உள்ள School data, Teachers data, students data, census data ஆகிய நான்கையும் முழுவதுமாக பூர்த்தி செய்து Save செய்து கொள்ளவும்.


2. working day's data ஒவ்வொரு மாத முடிவிலும் update செய்யவும்.


3. Marks data ஒவ்வொரு பருவத்தின் முடிவிலோ அல்லது அவ்வப்போதோ Update செய்து கொள்ளலாம்.


4. ஒவ்வொரு மாதமும் working day's data Update செய்த பிறகு forms button கிளிக் செய்து மாதாந்திர அறிக்கை download செய்து கொள்ளலாம். Admission form, Study certificate, Leave forms, Loan forms, என அனைத்து வகை படிவங்களையும் இப்பகுதியில் download செய்து கொள்ளலாம்.


5. Register பகுதியில் Marks data update செய்த பிறகு ஒவ்வொரு பருவத்திற்கும் CCE பதிவேடுகள் download செய்து கொள்ளலாம் தேவைப்படும் பொழுது இதர பதிவேடுகளையும் download செய்து கொள்ளலாம்.


6. Daily Homework பகுதியில் நாள் தேர்வு செய்து வகுப்பை தேர்வு செய்து Send கிளிக் செய்தால் மாணவர்களுக்கு அன்றைய நாளுக்கான வீட்டுப்பாடம் தானாக சென்று விடும். மாணவர்கள் அவர்களது login ல் நுழைந்த உடன் அன்றைய வீட்டுப்பாடம் திரையில் தெரியும். ஆசிரியர் பள்ளி நேரத்தில் வீட்டுப்பாடம் வழங்க முடியாமல் போனால், பேருந்தில் செல்லும்போது அல்லது வீட்டிற்கு சென்ற பிறகு கூட மாணவர்களுக்கு வீட்டுப் பாடத்தை அனுப்பி விட முடியும். இப்பகுதியில் ஒவ்வொரு நாளுக்கான வீட்டுப்பாட குறிப்புகள் ஏற்கனவே பதிவு செய்யப்பட்டிருக்கும். ஆசிரியர் விருப்பப்பட்டால் அதில் மாற்றம் செய்தும் மாணவர்களுக்கு அனுப்பி விடலாம்.


7. Daily messages பகுதியில் அந்த நாளுக்கான சிறப்புகள் பற்றிய குறிப்புகள் ஒவ்வொரு நாளுக்கும் இருக்கும். ஆசிரியர் காலை வழிபாட்டுக் கூட்டத்திலோ அல்லது வகுப்பறையின் போது கூடுதல் தகவல்களாக சொல்வதற்கு பயன்படுத்திக் கொள்ளலாம்.


8. Book back Q&A பகுதியில் ஒவ்வொரு வகுப்பிற்கும் ஒவ்வொரு பாடத்திற்கும் புத்தக கேள்விகளுக்கான விடைகள் இருக்கும். ஆசிரியர் Screen Cast செய்வதன் மூலம் Smart board அல்லது TV யில் பார்த்து மாணவர்கள் விடைகளை குறிப்பேட்டில் எழுதிக் கொள்ள பயன்படுத்திக் கொள்ளலாம்.


9. Lesson Plan பகுதியில் ஒவ்வொரு பாடத்திற்கான பாட குறிப்பினை பார்த்து பயன்படுத்திக் கொள்ளலாம்.


10. Exam பகுதியில் ஒவ்வொரு வாரத்திற்கான வாராந்திர தேர்வுகள் மற்றும் மாதாந்திர தேர்வு ஆகியவை கொடுக்கப்பட்டுள்ளது. இது மீத்திறன் மாணவர்களுக்கு அவர்களுடைய அறிவுத்திறனை அதிகரிக்க உதவும். Games பகுதியில் தமிழ், ஆங்கிலம், கணிதம் பகுதிகளில் அறிவுத்திறன் மற்றும் மொழித்திறனை வளர்க்கும் வகையிலான விளையாட்டுகள் நிறைந்த பகுதி இதை Screen Cast மூலம் அனைத்து மாணவர்களும் பங்கேற்று ஆர்வமுடன் விளையாட செய்வதன் மூலம் கற்றலில் ஆர்வத்தை ஏற்படுத்தலாம்.


11. G.Os பகுதியில் ஆசிரியர்களுக்கான சமீபத்திய அரசு ஆணைகள் Download செய்து கொள்ளலாம்.


12. Class Room என்ற தலைப்பின் அருகே வலது புறம் உள்ள Staff Room icon ஐ கிளிக் செய்வதன் மூலம் ஆசிரியர்கள் Social Media பகுதிக்கு செல்லலாம். இதில் தமிழகத்தில் உள்ள பெரும்பாலான ஆசிரியர்கள் அவர்களது கருத்துக்களை பதிவு செய்யும் வகையில் அமைக்கப்பட்டுள்ளது. மீண்டும் மேலே உள்ள Class room ஐகானை கிளிக் செய்தால் இதே பக்கத்திற்கு திரும்பி விடலாம்.


13. மாணவர்கள் login செய்யும்பொழுது login பக்கத்தில் student login என்ற Tab ஐ கிளிக் செய்து பள்ளியின் UDISE நம்பரை உள்ளீடு செய்து Search கொடுத்தால் பள்ளியின் பெயர் காட்டும் அதை சரி பார்த்த பிறகு அவர்களது சேர்க்கை எண்ணை User Name ஆகவும் பிறந்த நாளை Password ஆகவும் கொடுத்து login செய்தால் மாணவர்களுக்கான Dashboard பகுதியில் அன்றைய வீட்டுப்பாடம் இருக்கும் அதற்கு கீழ் தேர்வுகள், quiz, Games போன்ற option கள் இருக்கும். மாணவர்கள் வீட்டுப்பாடத்தினை தவறாமல் செய்யவும், ஓய்வு நேரத்தை பயனுள்ள வகையில் அறிவுத்திறனை வளர்க்கும்படி அமைத்துக் கொள்ளவும் இது பயன்படும்.


இது போன்ற அதிநவீன வசதிகளுடன் கூடிய மேம்பட்ட Software ஐ நீங்கள் பயன்படுத்த, பல ஆயிரங்கள் செலவு செய்ய வேண்டியிருக்கும். ஆனால் கல்விச் சேவையை முதன்மை நோக்கமாக கொண்ட நமது தளத்தில் அனைத்து வசதிகளையும் இலவசமாக பெற்று உங்கள் பணியினை மிகச் சிறப்பாக செய்து மாணவர்களின் கல்வித் திறனை வளர்த்திடுங்கள்.
